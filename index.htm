About HTML, Hypertext markup language

In 1986 a British scientist Tim Berners-Lee, the one who invented the World Wide Web (WWW), and HTML (HyperText Markup Language) originated around 1990, is parsed by the browser which renders the webpage to display.

If you need to structure a web page and its content, you will need to code using HTML, diferent of JAVA, JS, C++, C#, PY or any other programming language, html lacks variables, conditional statements or iterative loops, because of this, it is not a programming language.
